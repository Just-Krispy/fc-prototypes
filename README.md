## About

Built by **Krispy** and **Archer** for Greg Cunningham, CEO of Farm Credit SEMO — a $1B agricultural lending cooperative.

These prototypes demonstrate digital member experience tools designed to modernize Farm Credit operations and member engagement.

## Deployment

This repo auto-deploys to Vercel. Each push to `master` triggers a new deployment.

## Tech Stack

- Frontend: HTML/CSS/JavaScript (static, no frameworks)
- Hosting: GitHub Pages / Vercel
- Design: Mobile-first, accessible
