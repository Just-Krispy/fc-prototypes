# Good Morning SEMO - Executive Dashboard

Daily executive dashboard for Farm Credit Southeast Missouri leadership.

## Features
- Key lending metrics and portfolio overview
- Live commodity market prices
- Cape Girardeau weather with ag-relevant alerts (frost, heavy rain)
- Daily priorities and weekly performance stats
- Farm Credit official branding
- Mobile responsive

## Data Sources
- Weather: [wttr.in](https://wttr.in) API (Cape Girardeau, MO)
- Metrics: Demo data (will integrate real APIs in future)
- Market data: Demo data (will integrate real feeds in future)

## Tech Stack
Pure HTML/CSS/JS — no build step, no frameworks, fast loading.
